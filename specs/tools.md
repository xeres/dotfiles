# Tools

## mise

- When: chezmoi apply を実行する
- Then: `mise` コマンドが利用可能である

## Node.js

- When: mise install が完了する
- Then: `node` コマンドが利用可能である
- Then: `npm` コマンドが利用可能である

## uv

- When: mise install が完了する
- Then: `uv` コマンドが利用可能である

## GitHub CLI

- When: mise install が完了する
- Then: `gh` コマンドが利用可能である

## AWS CLI

- When: mise install が完了する
- Then: `aws` コマンドが利用可能である

## Starship

- When: mise install が完了する
- Then: `starship` コマンドが利用可能である
